# 마비노기 모바일 룬 챗봇 서버

마비노기 모바일 시즌1 룬 정보를 제공하는 카카오톡 챗봇 백엔드입니다.

## 📁 파일 구성

```
mabinogi_rune_bot/
├── app.py          # Flask 서버 (메인)
├── rune_data.py    # 룬 데이터 및 검색 로직
├── requirements.txt
└── README.md
```

## 🚀 설치 및 실행

### 1. 의존성 설치
```bash
pip install -r requirements.txt
```

### 2. 서버 실행
```bash
python app.py
```
기본 포트: **5000**

### 3. 배포 시 (gunicorn)
```bash
gunicorn app:app --bind 0.0.0.0:5000
```

## 🔌 카카오 i 오픈빌더 연동 방법

### Step 1. 서버 외부 공개
서버를 외부에서 접근할 수 있도록 배포합니다.
- **무료 옵션**: [Railway](https://railway.app), [Render](https://render.com), [Fly.io](https://fly.io)
- 또는 ngrok으로 로컬 테스트: `ngrok http 5000`

### Step 2. 카카오 오픈빌더 설정
1. [카카오 i 오픈빌더](https://i.kakao.com) 접속
2. 챗봇 생성
3. **스킬** 메뉴에서 새 스킬 추가
   - URL: `https://your-server.com/webhook`
   - 메서드: `POST`

### Step 3. 시나리오 설정
1. **시나리오** → 새 블록 생성
2. 패턴 등록: `/룬 *` (또는 사용자 발화 패턴)
3. 액션: 스킬 연결 → 위에서 만든 스킬 선택

### Step 4. 테스트
- 카카오톡에서 `/룬 낙화` 입력
- 룬 정보가 출력되면 성공!

## 🧪 직접 테스트 (API)

서버 실행 후 브라우저 또는 curl로 테스트:

```bash
# 룬 단건 검색
curl "http://localhost:5000/rune?name=낙화"

# 무기 전설 룬 목록
curl "http://localhost:5000/rune/list?category=무기&grade=전설"

# 전체 목록
curl "http://localhost:5000/rune/list"
```

## ➕ 룬 데이터 추가 방법

`rune_data.py`의 `RUNE_DATA` 딕셔너리에 추가:

```python
"룬이름": {
    "등급": "전설",          # 에픽 / 전설 / 신화 / 전설 ★8
    "분류": "무기",          # 무기 / 방어구 / 엠블럼 / 장신구 (클래스명)
    "효과": "효과 설명...",
    "획득": "레이드",        # 선택사항
    "별칭": ["룬이름+"],    # 선택사항 (검색 시 별칭으로도 찾을 수 있게)
},
```
